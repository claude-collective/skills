## Key Themes Across Resources

### 1. Distribution Solutions
- **Plugins** solve the distribution problem (youngleaders.tech)
- **Shai CLI** for configuration sharing (sebasjimenezvel)
- **bwc-cli** from buildwithclaude.com

### 2. Agent Categories
Most collections organize agents by:
- **Core Development:** Frontend, Backend, Full-stack
- **Quality/Security:** Code review, Security audit, Testing
- **DevOps:** CI/CD, Kubernetes, Cloud (AWS/GCP/Azure)
- **Data/ML:** Data engineering, ML pipelines
- **Documentation:** Technical writing, API docs

### 3. Context Management
- Isolated context windows prevent pollution (consistent theme)
- Agent-Memory-Protocol for coordination (Ilyas Ibrahim)
- Skills for efficient token usage vs CLAUDE.md always-loaded

### 4. Skill vs Subagent Pattern
- **Skills:** Auto-invoked, portable expertise, token-efficient
- **Subagents:** Explicit invocation, isolated context, specific tool access
